## Инструкция по запуску WebGL-игры в хабе Elytopia

Ниже описаны основные шаги, необходимые для корректного размещения и работы вашей WebGL-игры в хабе Elytopia.

---

## Шаг 1. Скачать и импортиUnity-пакета

1. Перейдите по ссылке для скачивания пакета:
   [Download latest Elytopia.unitypackage](https://site.to.download.our.package.or.github.link)
2. В уже открытом Unity-проекте откройте меню **Assets \> Import Package \> Custom Package**.
3. В появившемся окне **Import Unity Package** убедитесь, что все необходимые файлы выбраны, и нажмите **Import**.

---

## Шаг 2. Размещение игры

Перед размещением убедитесь, что интерфейс вашей игры соответствует требованиям:

- **В правом верхнем углу** необходимо оставить пустое пространство для кнопки закрытия игры.

  **Параметры кнопки:**
  1. Размер: `30 x 30` пикселей
  2. Отступы:
     - `20` пикселей от верхнего края экрана
     - `25` пикселей от правого края экрана

---

## Шаг 3. Взаимодействие с хабом через Singleton

Для коммуникации с хабом используется объект `KidsGameHubBridge.Instance`.

### Пример закрытия игры

```csharp
KidsGameHubBridge.Instance.SendToHub(GameHubCommand.CloseGame);
```

Команда `GameHubCommand.CloseGame` отвечает за закрытие игры.

### Пример показа рекламы (Interstitial)

```csharp
KidsGameHubBridge.Instance.SendToHub(GameHubCommand.ShowInterstitialAd);
KidsGameHubBridge.Instance.ShowInterstitialAd(values);
```

> **Важно\!**
> Частота показа рекламы регулируется хабом. Реклама **не должна** показываться чаще, чем через установленный интервал времени (N секунд).

**Примечания к командам:**

- `command` — команда, которую необходимо выполнить.
- `values` — дополнительные параметры, передаваемые вместе с командой (если требуются).

## Шаг 4. Распространение WebGL-сборки

Чтобы мы могли разместить вашу игру, нужно:

1. Заархивировать сборку **строго** в формате ZIP (никакие другие форматы не принимаются).
2. Использовать стандартный шаблон WebGL от Unity.
3. Соблюдать следующую структуру архива:

```plaintext
webgl_game_name_build.zip
│
├── Build/
├── index.html
├── ваши другие папки и файлы
└── TemplateData/ (эту папку можно удалить)
```

---

## Шаг 5. Добавление игры в каталог

Для добавления игры в каталог необходимо заполнить форму, указав:

- **Имя игры**: `ВАШЕ_НАЗВАНИЕ_ИГРЫ`
- **ID игры / Bundle**: `com.game.bundle`
- **Версия**: `0.1`

Затем прикрепить демонстрационное изображение, которое будет отображаться в каталоге.

**Размер изображения:** `256 x 167` пикселей.
