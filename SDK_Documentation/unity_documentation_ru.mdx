## Шаг 1. Скачать и импортировать SDK

1. Перейди по ссылке для скачивания пакета

   <Card title="Elytopia-SDK Unity" href="https://github.com/Elytopia-Studio/ElytopiaSDK-Unity">
     
   </Card>
2. В Unity-проекте с игрой открой меню **Assets \> Import Package \> Custom Package** и выбери скаченный `elytopia-sdk.unitypackage`
3. В появившемся окне **Import Unity Package** убедись, что все необходимые файлы выбраны, и нажмите **Import**

---

## Шаг 2. Показ рекламы

<Tip>
  Реклама запускается не при каждом показе рекламы. Система Elytopia сама определяет, когда пользователь готов к просмотру новой рекламы, поэтому не стесняйтесь указывать как можно больше вызовов рекламы
</Tip>

### Interstitial реклама

Это не долгая, полноэкранная видео-реклама

```csharp
using Elytopia;

ElytopiaSDK.Instance.ShowInterstitialAdAsync();
```

## Шаг 4. Размещение готовой сборки

Чтобы мы могли разместить вашу игру, нужно:

1. Заархивировать сборку **строго** в формате ZIP (никакие другие форматы не принимаются)
2. Использовать стандартный шаблон WebGL от Unity
3. Соблюдать следующую структуру архива:

```plaintext
webgl_game_name_build.zip
│
├── Build/
├── index.html
├── Другие папки и файлы
└── TemplateData/ (эту папку можно удалить)
```

---

## Шаг 5. Добавление игры в каталог

Для добавления игры в каталог необходимо согласовать с личным менеджером следующие пункты

- **Название игры**: `Game Name`
- **Bundle**: `com.game.bundle`
- **Версия**: `0.1`
- **Обложка:** `256 x 167` пикселей
